## 🎨 Cyberpunk RGB Neon UI
* **Premium Glassmorphic Design:** Glowing animated RGB dynamic borders ke sath modern transparency wrapper.
* **Zapya-Style Camera Scanner:** Bina ID copy-paste kiye, back-camera lens se direct dusre phone ka QR code scan karke auto-pairing.
* **Real-time Dual Sync Percentage:** Bhejne wale aur receive karne wale, dono ki screen par live transfer speed progress status tracker.

---

## 🛡️ Security Features (Hacker-Proof Tunnel)
1.  **End-to-End Encryption (E2EE):** Data peer-connection ke andar completely encrypted form mein transfer hota hai, jisse Man-in-the-Middle attacks se safety milti hai.
2.  **Anti-Malstream Packet Drop:** Sockets par unauthenticated or corrupted injections ko shield filter instantly block karke disconnect kar deta hai.
3.  **Isolated Sandbox:** Sari incoming files RAM memory chunks mein process hoti hain, jisse aapke local device ki storage safe rehti hai.

## 🚀 Speed Benchmarks
Traditional servers (Google Drive, WhatsApp) par pehle file upload hoti hai fir download hoti hai. SonicShare direct link banata hai:
* **Any Extension Allowed:** `.mp4, .apk, .zip, .exe, .pdf, .png` etc.
* **256KB Dynamic Chunk Engine:** Local Wi-Fi router ki maximum capacity utilize karke **up to 80MB/s+** tak ki speed deliver karta hai.
